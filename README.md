## ğŸ› ï¸ Installation & Setup

### Prerequisites

- Node.js 18.17.0 or later
- npm, yarn, or pnpm

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/diptadeep21/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)


### Update Projects

Edit `utils/data/projects-data.js` to add or modify your projects.

### Update Skills

Edit `utils/data/skills.js` to customize your skills list.

### Update Experience & Education

- Edit `utils/data/experience.js` for work experience
- Edit `utils/data/educations.js` for education background
